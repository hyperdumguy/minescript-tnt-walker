# 💣 TNT Trail (Minescript)

I made this little Minescript for fun—it just places TNT under you every second while you're playing. That’s literally it. It turns your entire run into a disaster zone.

It’s simple, chaotic, and way more entertaining than it should be.

## What it does
Places TNT directly beneath you every second
Keeps going as long as the script is running
Works while you’re moving, jumping, running… doesn’t matter
How to use it
Make sure you have Minescript set up
Download or clone this repo
Put the script into your Minescript scripts folder
Run it in-game

Once it’s running, TNT will start spawning under you automatically.

## Fair warning

This will absolutely destroy your world if you let it run long enough.
Don’t use it on anything you care about unless you’ve made a backup.

Seriously.

### Why I made this

Mostly just to mess around and see how chaotic things could get. It’s actually pretty fun for:

Challenge runs (don’t stop moving)
Messing with friends in multiplayer
Watching everything slowly explode behind you(if I add something like this)
